Completed Visualization

Which Neurological Conditions Cause the Most Unstable Walking Patterns?

This box plot visualizes the standard deviations of left stride intervals for patinets in different disease groups (ALS, Huntington’s, Parkinson’s, and Control). A higher standard deviation indicates greater variability in stride, which correlates with instability in gait. Values closer to 0 indicate a more balanced gait.

The x-axis is on a log scale to better represent the wide range of standard deviation values while also preserving relative differences.

Hover over the dots to learn more details about the exact patient or hover over the boxplot to learn more about the distribution of each disease group.

How Does the Gait Symmetry Index Differ in Individuals with Neurodegenerative Diseases and what does that indicate?

A shoe icon can dynamically represent gait asymmetry by showing how the left and right foot are moving in relation to the other, whether it is more left-aligned or right-aligned. This would provide an intuitive and visual understanding of gait imbalances, which could be crucial for early diagnosis and treatment evaluation. The shoe icon would move in sync with recorded gait data. It would display weight distribution and alignment during walking like if left foot dominates, the icon leans left, if right foot dominates, the icon leans right, If balanced, it stays neutral. Using stride intervals, stance intervals, and swing intervals, we can calculate a gait symmetry index (GSI) and show it as an interactive feature like a tooltip when we hover over the shoe icons GSI = (2 * | Left - Right |) / (Left + Right) Higher values indicate greater imbalance and lower values indicate more symmetrical gait.
